# Technology stack:
### Backend:
- Java
- Spring boot
- Spring Data REST
- Spring Data JPA
- Spring Security
- PostgreSQL database 

### Frontend:
- Angular 7
- Nebular framework for angular
- Various PrimeNG components for angular
- Leaflet
- Leaflet Routing Machine
- Leaflet Control Geocoder

### Other:
- Arduino IDE for GPS tracker
- Mapbox API account
